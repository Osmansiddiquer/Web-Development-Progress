Date and Time:

Try using Date.now(), and Date.parse() in the console. JS provides setter and getter functions for all the properties of the Date object. MDN has really good documentation on it.




Math Object:

The Math namespace object contains static properties and methods for mathematical constants and functions. Math works with the Number type. It doesn't work with BigInt.


Unlike most global objects, Math is not a constructor. You cannot use it with the new operator or invoke the Atomics object as a function. All properties and methods of Math are static.


Look at the documentation for the complete lists of properties(constants), and methods(functions) in the Math Object. It is important to realize that JavaScript does not have classes, and is instead a prototype-based OOPL.


You can also type 'Math' in the console to get the list of members in the object.
