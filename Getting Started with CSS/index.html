Starting with CSS!

Did you know: CSS stands for Cascading Style Sheets
Inline-CSS:

This is a paragraph syled using inline CSS: Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam atque optio blanditiis facilis maiores fugit! As you can see, there is no seperate CSS code.

Source code:
<p style="color: blue; border-radius: 5px;border: solid 2px green; background-color:rgb(127, 181, 118);"> This is a paragraph syled using <strong>inline CSS</strong>: Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam atque optio blanditiis facilis maiores fugit!</p>


Internal/Embedded-CSS:

This is a paragraph syled using internal/embedded CSS: Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam atque optio blanditiis facilis maiores fugit!

Source code:
<p> This is a paragraph syled using <strong>internal/embedded CSS</strong>: Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam atque optio blanditiis facilis maiores fugit!</p>
Embedded-CSS:
<style> p { color: green; border-radius: 5px; border: solid 2px green; background-color:rgb(165, 191, 50); } </style>


External-CSS:

This is a paragraph syled using external CSS: Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam atque optio blanditiis facilis maiores fugit!

Source code:
<p class="external_paragraph"> This is a paragraph syled using <strong>external CSS</strong>: Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam atque optio blanditiis facilis maiores fugit!</p>
External-CSS:
.external_paragraph { color: rgb(138, 18, 168); border-radius: 5px; border: solid 2px rgb(166, 27, 124); background-color:rgb(224, 155, 17); }
Notes:
  • See the source code to find other instances of inline-CSS
  • The CSS that is written later takes precedence unless marked with the "!important' tag (like in line 12)
